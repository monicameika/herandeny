# Imágenes

Añadir aquí las fotos de Herandeny.

## Archivos necesarios

| Archivo | Dimensiones | Uso |
|---|---|---|
| `herandeny.jpg` | Vertical, mín. 800px ancho | Sección "Sobre mí" |
| `og-image.jpg` | 1200×630px | Compartir en redes sociales |

## Consejos para la foto principal

- Fondo neutro o en tonos crema/terracota
- Buena iluminación natural
- Formato vertical (retrato)
- Que transmita calidez y profesionalidad
